About Streaming Lists X

Streaming Lists is a simpler way to find something worth watching without bouncing between streaming apps, search results, and social media lists.

The app pulls together popular movies and TV recommendations from major platforms including Netflix, Prime Video, Disney+, Hulu, HBO Max, and Apple TV+.

It is built with Angular 22, uses @angular/ssr for server-side rendering, and includes service worker support for Progressive Web App behavior.

Data is provided by The Movie Database (TMDb). This product uses the TMDb API but is not endorsed or certified by TMDb.

Built by Ian Poston in Los Angeles.
